# Nakamura Profile Site

## サイト概要

中村の自己紹介を目的とした、柔らかく明るい雰囲気のプロフィールサイトです。赤をメインカラーにし、情報を見つけやすい統一デザインにしています。

## ページ構成

- `index.html`：サイトの入口、仮テキスト、ページ導線、画像スライダー
- `about.html`：名前、専攻、自己紹介、将来の目標、強み
- `skills.html`：スキルと資格をカード形式で紹介
- `favorite.html`：好きなものをカード形式で紹介
- `contact.html`：送信機能のないお問い合わせフォーム

## 使用技術

- HTML5
- CSS3
- Vanilla JavaScript
- 自作SVG

## 実装したJavaScript機能

- ハンバーガーメニュー
- TOPページの画像スライダー
- Contactページのフォーム入力チェック
- ページ上部へ戻るボタン
- スクロール時のフェードイン表示

## デザインの工夫

柔らかな赤、淡い背景色、角丸、控えめな影を共通化しました。全ページでヘッダー、ナビゲーション、フッターを統一し、スマートフォン表示では1カラム中心に切り替わります。

## 著作権について

画像は本サイト用に作成したオリジナルSVGです。外部画像や第三者の文章・コードは使用していません。Adoの名称は好きなものを示す目的でのみ記載し、画像や歌詞などの著作物は使用していません。

## 生成AIによるコードレビュー

生成AIによるコードレビューを行い、HTML構造、CSSの共通化、JavaScriptの要素存在確認、レスポンシブ対応、アクセシビリティ、ファイル構成を確認しました。

## お問い合わせフォーム

お問い合わせフォームは入力チェックのみを実装しています。送信機能はありません。

## コードの記述形式

すべてのHTML、CSS、JavaScript、SVGは、VS Codeで読みやすいように改行とインデントを付けて記述しています。
